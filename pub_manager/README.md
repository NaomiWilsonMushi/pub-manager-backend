## Pub Manager Backend API

### Overview  
This project is a backend API built to help manage the daily operations of a pub.  
It focuses on handling drink inventory, categories, orders, and basic stock tracking in a clean and structured way.

The idea comes from real-life pub management needs such as knowing what drinks are available, tracking stock, and recording orders efficiently.

---

### Features  
- Drink category management (Beer, Wine, Spirits, Soft Drinks)  
- Drink and inventory management  
- Order creation and tracking  
- Admin and staff role separation  
- Permission-based access control  
- RESTful API design  

---

### Tech Stack  
- Python  
- Django  
- Django REST Framework  
- SQLite (development)  

---

### Project Structure  
The project is structured to keep the code clean and maintainable:

- `pub_manager/` – main project settings  
- `bar/` – core app for drinks, categories, and orders  
- `manage.py` – Django management script
